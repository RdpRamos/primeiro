Hospedagem de site

A hospedagem de um site, consiste em deixá-lo disponível em um servidor web para que qualquer usuário da internet que tenha a URL (endereço web) poderá acessar o site.

O Githubpages é uma alternativa gratuita para hospedagem de sites!

Branches

São ramificações de um projeto, quando criamos ramificações secundarias permite-se alterações do código em paralelo, sem alterar o braço principal (branche main)
